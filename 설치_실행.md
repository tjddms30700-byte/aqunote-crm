# 🌊 AQUNOTE - 원장님 1분 설치 가이드

## 📌 3단계로 웹앱 띄우기

### ① Supabase 회원 데이터 이관 (5분)
1. https://supabase.com/dashboard/project/ngewuwxrvhorsfrdxlfu/sql/new 접속
2. `AQUNOTE_MEMBERS_IMPORT.sql` 다운로드 후 전체 복사 → 붙여넣기 → **Run**
3. 결과: `아동 27, 성인 19, 전체 46, 조직 1, 활동 26, 시간표 60, 평가 2` ← 이렇게 나오면 성공!

### ② 로컬 실행 (Node.js 18+ 필요)
```bash
cd aqunote_web
npm install
npm run dev
```
브라우저에서 http://localhost:3000 열기

### ③ Vercel 배포 (5분)
1. GitHub 계정으로 https://vercel.com 로그인
2. `aqunote_web` 폴더를 GitHub 새 저장소로 업로드
3. Vercel에서 "Add New Project" → 저장소 import
4. **Environment Variables** 섹션에서 아래 2개 추가:

| 이름 | 값 |
|---|---|
| `NEXT_PUBLIC_SUPABASE_URL` | `https://ngewuwxrvhorsfrdxlfu.supabase.co` |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | `sb_publishable_K-qxr6wIYdRQtYvupY_m4A_82w2zNwE` |

5. **Deploy** 클릭 → 30초 후 `aqunote.vercel.app` 완성!

## ❓ 문제가 생기면
- Node.js 미설치: https://nodejs.org 에서 LTS 버전 다운로드
- npm install 오류: 프로젝트 폴더에서 `rm -rf node_modules && npm install` 재실행
- 회원이 0명으로 표시: SQL이 아직 안 돌아간 상태 → ① 단계 다시 확인
